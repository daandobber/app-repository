# Kodi Remote

Native Kodi remote control for the Nicolai Electronics Tanmatsu and Konsool.

Source repository: https://github.com/daandobber/kodiremote

## Features

- Directional, playback, seeking and volume controls.
- Rich now-playing view with cover art, media details and timeline.
- Browser for movies, TV shows and music in the Kodi library.
- On-device text entry for Kodi search fields.
- Optional library and cover-art preloading.
- Configurable sleep after 30, 60 or 90 seconds switches off the display,
  keyboard backlight and status LEDs together.
- Stores the Kodi host, port and optional login details on the device.

## Setup

Connect the Tanmatsu or Konsool to the same WiFi network as Kodi. In Kodi,
enable **Settings > Services > Control > Allow remote control via HTTP**. Enter
that host, port and any configured username and password in the app settings.

Press `F1` to return to the launcher. The full controls and build instructions
are documented in the source repository.
